# TodoListAngular

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 15.1.6.

## Inicianizar el proyecto

Para hacer correr este proyecto tendrás que seguir los siguientes pasos:
-> Clonar el proyecto
-> Dentro de las carpeta ya clonada realizar un `npm i`
-> Abrir dos terminales:
   -> La primera hacer un `npm run server` para inicializar el json-serve
   -> La segunda hacer un `ng serve` para ver el front del proyecto
-> Testear la app introduciendo tus tareas
